# To-do List

## 📌 Descrição
Projeto desenvolvido para praticar 
Cria uma lista de tarefas na qual quando concluidas
ao apertar o botão ela risca a atividade ou voce pode excluir.

## 🚀 Tecnologias
- HTML
- CSS
- JavaScript
- Node.js

## ⚙️ Funcionalidades
- Cadastro de dados
- Listagem
- Interação com usuário

## ▶️ Como executar
1. Baixe o projeto
2. Abra o arquivo index.html
